# Guia Farmácia ONG

Este projeto é uma aplicação web desenvolvida em React que serve como um **guia de medicamentos** para auxiliar no atendimento de pacientes em uma farmácia de ONG. O sistema facilita a busca por medicamentos e fornece informações importantes sobre dosagens, indicações e cuidados necessários.

## 📋 Funcionalidades

### 1. Busca de Medicamentos

- Interface rápida para encontrar medicamentos por nome.
- Ordenação alfabética automática dos resultados.

### 2. Ficha Detalhada do Medicamento

Ao selecionar um medicamento, o usuário tem acesso a:

- **Nome comercial e princípios ativos**
- **Nome técnico**
- **Apresentações** disponíveis
- **Classe terapêutica** (ex: Anti-hipertensivo, Diabético, etc.)
- **Forma farmacêutica** e **via de administração**
- **Indicações**
- **Dosagem padrão**
- **Precauções e contraindicações**
- **Informações de segurança** relevantes

### 3. Temas

O sistema oferece três modos de visualização para atender diferentes preferências:

- **Claro** (Light)
- **Escuro** (Dark)
- **Industrial**

## 🛠️ Tecnologias Utilizadas

- **React**: Biblioteca principal para construção da interface.
- **JavaScript (JSX)**: Lógica e marcação da aplicação.
- **HTML5 + CSS3**: Estilização e layout.

## 🚀 Como Executar

### Pré-requisitos

Certifique-se de ter um ambiente Node.js configurado. Este projeto utiliza importações dinâmicas de módulos React, portanto, **não** requer um processo de build tradicional com Webpack ou Vite para execução local, bastando abrir o arquivo `index.html` no navegador.
